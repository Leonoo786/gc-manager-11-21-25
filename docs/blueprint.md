# **App Name**: ConstructionFlow

## Core Features:

- Dashboard: Displays key project metrics and recent activities, linking to detailed pages.
- Project Management: CRUD operations for projects with detailed views including budget, tasks, documents, photos, team, contractors and vendors.
- Budget Management: Comprehensive budget tracking with summaries, categories, transactions, analytics, and forecasting.  Includes export and import functionalities for budget data.
- Schedule and Task Management: Task management with Kanban board and list views, including dependencies and bulk actions.
- Document and Photo Management: Upload, preview, tag, and manage project documents and photos.
- Team and Vendor Management: Manage project teams, contractors, and vendors with roles, contact info, and CRUD operations.
- Profit & Loss Reporting: Generate a visual summary of profitability, considering bids, planned budget, actual expenses, and estimated profit. The tool will decide which inputs to incorporate in its outputs based on data quality

## Style Guidelines:

- Primary color: Deep blue (#2E4A62) evoking stability and trust, for the primary UI elements.
- Background color: Light gray (#F0F4F8) for a clean, professional backdrop that ensures readability.
- Accent color: Orange (#E07A5F) to highlight key interactive elements such as calls to action.
- Headline font: 'Space Grotesk' sans-serif for a modern, techy feel, ideal for headlines.
- Body font: 'Inter' sans-serif for a clean, readable style in longer text blocks.
- Consistent use of line icons throughout the app for wayfinding and visual interest.
- Desktop-first responsive design with clear content hierarchy and intuitive navigation, leveraging cards and tables for data presentation.
